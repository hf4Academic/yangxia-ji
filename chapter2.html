📚 Skills是什么？怎么创建？

🎯 Skills 的组成部分

📌 Name（名称） - 给技能起个名字，让大家知道这个技能叫什么

📌 Description（描述） - 告诉AI什么时候该用这个技能

📌 Tools（工具） - 这个技能可以使用哪些工具（搜索、读文件、发邮件等）

📌 Instructions（指令） - 详细的使用说明，AI按照这个来执行

📌 Examples（示例） - 给几个例子，让AI知道用户会怎么问

✏️ 创建一个Search Skill的例子

## Search Skill

**name**: search  
**description**: 当用户需要搜索信息、查找资料时使用  
**tools**: web_search, browser_open  
**instructions**: |
  1. 理解用户想搜索什么
  2. 使用web_search工具搜索相关内容
  3. 整理搜索结果，提取关键信息
  4. 用简洁的语言回答用户

**examples**: |
  - "今天天气怎么样"  
  - "帮我查一下北京到上海的机票"
  - "最新的人工智能新闻"

💡 就像给小龙虾写了一本使用手册，告诉它：
- 这个技能叫什么
- 什么时候用
- 可以用什么工具
- 怎么使用
- 用户会怎么问

🎨 真实例子：创建PPT的Skill

下面是一个真实使用的Skill，来自我的技能库。让我把它拆开，一块一块来解释： 

---
name: powerpoint
description: "Create, read, edit .pptx decks, slides, notes, templates."
---

## Quick Reference
| Task | Guide |
|------|-------|
| Read/analyze | python -m markitdown presentation.pptx |
| Edit | Read editing.md |
| Create | Read pptxgenjs.md |

## When to use
Use this skill any time a .pptx file is involved — creating slides, reading presentations, editing templates... Trigger when user says "deck", "slides", "presentation", or mentions .pptx.

## Tools
- markitdown (extract text)
- pptxgenjs (create slides)
- python-pptx (edit existing)
- GPT Image 2 (generate images for slides)

## Instructions
1. 分析需求：理解用户要什么样的PPT
2. 选择配色：根据主题选颜色搭配
3. 生成图片：需要图片时用GPT Image 2生成
4. 创建/编辑：用pptxgenjs从头创建，或编辑模板
5. QA检查：转成图片检查是否有问题

## Examples
- "帮我做一个产品介绍PPT"
- "，把这 个PDF转成PPT"
- "修改第3页的标题"
🔍 逐块解读
1️⃣ name: powerpoint
就像给技能起名字，这是"创建PPT"的技能
2️⃣ description
告诉AI：只要是跟PPT相关的任何操作都用这个技能（创建、读取、编辑）
3️⃣ When to use
详细说明什么情况下触发：用户说"deck"、"slides"、"presentation"，或提到.pptx文件
4️⃣ Tools 工具列表
• markitdown - 读取PPT内容
• pptxgenjs - 创建新PPT
• python-pptx - 编辑现有PPT
• GPT Image 2 - 生成幻灯片配图
5️⃣ Instructions 步骤
1️⃣ 分析需求 → 2️⃣ 选配色 → 3️⃣ 生成图片 → 4️⃣ 创建PPT → 5️⃣ QA检查
6️⃣ Examples 示例
告诉AI用户可能的问法，这样AI就能准确理解用户意图

💡 总结：一个完整的Skill就像一份完整的产品说明书，让AI知道：
  ✔ 这个技能叫什么
  ✔ 什么时候用
  ✔ 可以用什么工具
  ✔ 具体怎么操作（步骤）
  ✔ 用户通常会怎么说
