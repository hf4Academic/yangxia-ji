## 第1章：初始形态 - CLI对话

### 🎯 主题
只能在命令行里对话的虾苗

### 🎨 视觉设计
- 复古Terminal/命令行界面
- 绿色文字 on 黑色背景
- 闪烁的光标 _
- 虾苗很小，在命令行里

### 📋 内容结构
1. **开场动画** - 屏幕 slowly appear，显示 "C:\> _"
2. **虾苗介绍** - "你好，我是一只小虾苗"
3. **能力展示** - 演示基本对话（只能打字）
4. **限制说明** - 只能坐电脑前，不能动
5. **对比** - vs 后面的阶段有多不方便

### 🎮 交互
- 点击命令行可以输入问题
- 简单对话演示

### 📝 文字内容
```
我是虾小一 🦐
我刚出生，还很小很小...
我只能听懂命令行里的指令
你必须坐在电脑前，敲键盘和我说话
我不能动，也不能去别的地方
但是！我正在学习，也许有一天能变得更厉害~
```

## 视觉风格统一

### 配色方案
- 主色： Ocean Blue `#0ea5e9`
- 辅色： Mint Green `#10b981`  
- 警告色： Alert Red `#ef4444`
- 背景： Deep Ocean `#0f172a`

### 动画效果
- 页面切换：渐隐渐显
- 虾的状态：游动/进化动画
- 任务完成：星星飞散

### 字体
- 标题：圆体/幼圆
- 正文：思源黑体

## 技术实现

- 单页应用，点击切换章节
- 使用 CSS 动画
- 响应式设计（手机/电脑都能看）
- 每个章节独立HTML文件便于维护
